What is QuickSWE?

🎯 Purpose

  • Fast, rigorous benchmark for AI coding agents
  • 100 tasks across 3 languages, 4 difficulty tiers
  • Bug fixes, features, and refactoring

⚡ Practical

  • No Docker — runs on your laptop
  • Under 100MB storage (vs 200GB for SWE-Bench Pro)
  • Built-in statistical multi-run analysis

📏 Rigorous

  • Every measurement backed by real pytest
  • No LLM-as-judge, no subjective scoring
  • Tests pass or they don't

🔴 Hard

  • Red-black trees, compilers, VMs, GCs
  • Lock-free data structures, coroutines
  • CUDA dynamic parallelism, warp BFS, FFT
2 / 10

Methodology

🔬 Test Protocol

  • Agent gets task description + buggy source code
  • Must fix code to pass fail_to_pass tests
  • Existing pass_to_pass tests must NOT break
  • Wall-clock time measured per task

📊 Statistical Rigor

  • 3 independent runs per agent
  • Agents alternate order each round
  • Consistency score per task
  • Aggregate resolve rates with variance

🛡️ Safety Measures

  • NTFS deny-delete ACLs on critical directories
  • Guardrail prompts (CLAUDE.md / AGENTS.md)
  • Post-task directory integrity checks
  • Isolated temp workspaces per task

⚖️ Fairness

  • Same prompt, same timeout (600s)
  • Same machine, same environment
  • Both agents use bypass-permissions mode
  • Latest versions (April 2026)

Key Findings

  1. Amp Deep³ wins across all three languages — Python, C++, and CUDA
  2. The gap widens with language complexity: Python (+1%) → CUDA (+16%) → C++ (+46%)
  3. Near-zero regressions: Amp breaks existing tests 13x less often than Claude (0.4% vs 5.2%)
  4. Perfect consistency: Amp resolves the exact same tasks across all 3 runs on Python and CUDA
  5. Claude hit API rate limits during testing — a real-world reliability concern for intensive workloads
  6. Amp trades speed for accuracy: slower per-task but significantly higher resolve rate, especially on hard tasks
